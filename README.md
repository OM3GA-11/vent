## 📖 Overview

V.E.N.T is a full-stack anonymous emotion sharing platform built using the MERN ecosystem (with PostgreSQL instead of MongoDB). Users can securely register, create emotion-based posts, vote, comment, and browse the community feed.

The application focuses on clean architecture, responsive design, secure authentication, and efficient server-state management.

---

## ✨ Features

- 🔐 JWT Authentication
- 👤 User Registration & Login
- 📝 Create Anonymous Vents
- 😊 Emotion-based Filtering
- 👍 Upvote & Downvote System
- 💬 Comment System
- 👤 User Profiles
- 📱 Fully Responsive Design
- ⚡ React Query Data Fetching
- 💾 Persistent Login using Zustand
- 🌐 Cloud Deployment (Vercel + Render + Neon)
